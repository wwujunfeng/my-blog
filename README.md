# MyBlog

#### 介绍
个人Blog - Vue 3 + Vite 博客项目

#### 软件架构
- Vue 3 + Vite
- Bootstrap 4.3.1
- Font Awesome 4.7.0
- 响应式设计，支持桌面、平板和手机设备

#### 安装教程

1.  克隆仓库：`git clone https://gitee.com/wujunfeng-develop/my-blog.git`
2.  进入项目目录：`cd my-blog`
3.  安装依赖：`npm install`
4.  启动开发服务器：`npm run dev`
5.  构建生产版本：`npm run build`

#### 使用说明

1.  开发服务器运行在 http://localhost:5173/
2.  项目包含首页、关于、作品集、博客和联系等模块
3.  支持平滑滚动、悬停效果和滚动触发动画
